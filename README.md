# 💼 Finora — Finance Dashboard  
### Zovryn Frontend Assessment Submission

Finora is a modern, interactive **Finance Dashboard Web Application** built as part of the Zovryn Frontend Assessment.

The application focuses on delivering a clean user experience, real-time financial insights, and a scalable UI architecture using modern frontend technologies.

---

## 🚀 Overview

This project demonstrates:

- Component-based architecture using React  
- Clean and scalable UI design  
- State management using Context API  
- Interactive charts and analytics  
- Smooth animations and transitions  
- Dark/Light theme system  

---

## ✨ Key Features

- 📊 Dashboard overview (Balance, Income, Expenses)
- 📈 Insights with analytics
- 💳 Transaction management (Add / Edit / Delete)
- 📉 Category-based expense visualization (Pie Chart)
- 📅 Transaction history table
- 🔔 Notification system
- 👤 User profile and settings
- 🌙 Dark / Light mode with persistence
- ⚡ Smooth UI animations (Framer Motion)

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React Icons
- Context API

## ⚙️ Installation

git clone https://github.com/Shekhar6264/Finora.git  
cd Finora 
npm install  
npm run dev  

---

## 🌙 Theme Implementation

- Global theme handled via React state  
- Applied using Tailwind's `dark` class  
- Persisted using `localStorage`  

---

## 🎯 Design Approach

The UI is designed to balance:

- Clean usability (SaaS-style layout)
- Modern aesthetics (glass + gradients)
- Performance (lightweight rendering)
- Scalability (reusable components)
